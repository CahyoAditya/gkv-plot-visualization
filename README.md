# GKV Plot Visualization

This repository contains a **Computer Graphics and Visualization** task/project.

## Project Brief
The assignment is to **pick a dataset** and **visualize it using Plotly.js** (JavaScript).

For this project, the chosen dataset is:
- **Video Game Sales and Industry Data (1980–2024)** (Kaggle)
- https://www.kaggle.com/datasets/bhushandivekar/video-game-sales-and-industry-data-1980-2024

## Goal
Build interactive visualizations (charts/plots) with **Plotly.js** to help explore trends and insights in the selected dataset.

## Tech Stack
- **Plotly.js** for interactive visualizations
- **JavaScript / HTML** for the visualization pages
- **Python** may be used for data cleaning/preprocessing (optional)

## Notes
- The dataset files are downloaded from Kaggle and then used as the data source for the plots.
- Visualizations should be interactive (hover tooltips, zoom/pan, filters, etc.) where possible.
